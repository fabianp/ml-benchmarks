\title{Benchmark for several Python machine learning packages}

\maketitle
We compare computation time for a few algorithms implemented in the
major machine learning toolkits accessible in Python. We use the
Madelon data set \citep{Guyon2004}, 4400 instances and 500 attributes,
that can be used in supervised and unsupervised settings and is quite
large, but small enough for most algorithms to run.
